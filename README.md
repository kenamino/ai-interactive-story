# ✨ AI 互动小说生成器

一个基于 AI 的互动式小说生成器，让你成为故事的创造者。每一次选择，都将开启一段独特的冒险。

![Project Banner](https://img.shields.io/badge/AI-Powered-blue) ![License](https://img.shields.io/badge/license-MIT-green) ![OpenAI](https://img.shields.io/badge/OpenAI-GPT--4-orange)

## 🎯 项目特点

- **AI 驱动**：使用 OpenAI GPT-4 模型实时生成故事内容
- **多种类型**：支持科幻、奇幻、悬疑、浪漫、恐怖、武侠等多种故事类型
- **自定义开局**：可以自己定义故事的开始
- **真实选择**：你的每个选择都会真正影响故事走向
- **精美界面**：现代化的深色主题界面，打字机效果展示
- **完全本地**：API Key 保存在浏览器本地，保护隐私
- **响应式设计**：支持桌面和移动设备

## 🚀 快速开始

### 在线体验

访问 [GitHub Pages 部署地址](#) 即可直接使用（需要自己的 OpenAI API Key）

### 本地运行

1. 克隆项目
```bash
git clone https://github.com/你的用户名/ai-interactive-story.git
cd ai-interactive-story
```

2. 直接打开 `index.html` 文件即可使用（无需安装任何依赖）

3. 输入你的 OpenAI API Key，开始创作你的故事！

## 🎮 使用方法

1. **配置 API Key**
   - 在欢迎页面底部输入你的 OpenAI API Key
   - API Key 会安全地保存在浏览器本地存储中

2. **选择故事类型**
   - 点击预设的故事类型按钮（科幻、奇幻等）
   - 或者在文本框中输入自定义的故事开局

3. **做出选择**
   - AI 会生成故事情节，并在关键节点提供 2-3 个选择
   - 点击你想要的选择，故事会根据你的决定继续发展

4. **享受冒险**
   - 每个故事都是独一无二的
   - 可以随时重新开始，体验不同的故事线

## 🛠️ 技术栈

- **前端**：纯 HTML + CSS + JavaScript（无框架依赖）
- **AI**：OpenAI GPT-4o-mini API
- **样式**：现代化 CSS3，渐变、动画、响应式设计
- **存储**：LocalStorage（本地存储 API Key）

## 📝 获取 API Key

1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 注册/登录账号
3. 进入 API Keys 页面
4. 创建新的 API Key
5. 复制并粘贴到应用中

**注意**：API Key 是敏感信息，请勿分享给他人。本应用将 Key 保存在浏览器本地，不会上传到任何服务器。
